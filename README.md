# CleanBoard

CleanBoard is a lightweight Windows 10/11 & MacOS app that removes any formatting or special characters from your copied text by hitting the copy-shortcut twice.

## About

CleanBoard helps you copy and paste plain text without any unwanted formatting. Simply copy text twice using your usual copy shortcut (Ctrl+C or Cmd+C) and CleanBoard will automatically clean the text, removing all formatting and special characters.

## Features

- Lightweight and efficient
- Works on Windows 10/11 and macOS
- Simple double-copy activation
- Removes text formatting and special characters
- Easy integration with your existing workflow

## Pricing

CleanBoard is free for up to 5,000 cleaned characters. After reaching this limit, a license purchase is required to continue using the app.
